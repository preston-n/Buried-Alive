# Buried Alive
  * **Trailer:**  https://www.youtube.com/watch?v=Gf6BLAZR3XA
  * **Description:** A 2D platformer game in which you must dig, build and use magic to survive
  * **Role:** Programmer
  * **Some of the worked on features:**
    * FMOD Audio
    * UI
    * Scoring System 
    * Player mechanics
  * *Made in C using Alpha Engine*
